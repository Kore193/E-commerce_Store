
##🛍️ Simple E-Commerce Store (TechZone)

# 📌 Overview
This is a full-stack e-commerce application built using:
- **Frontend:** HTML, CSS, JavaScript, Bootstrap  
- **Backend:** Node.js + Express.js  
- **Database:** MongoDB Atlas  

# ⚙️ Features
✅ User Registration & Login (JWT Authentication)  
✅ Product Listings with Categories  
✅ Shopping Cart (Add, Remove, Clear)  
✅ Place Order (Saved in MongoDB)  
✅ View Orders by Logged-In User  
✅ Responsive Bootstrap UI (TechZone Theme)
